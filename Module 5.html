Launching into Cyber Security

Cyber Security is now a global issue affecting society at all levels. The ever-increasing reliance on the Internet and the World Wide Web for almost every sphere of society from e-commerce to healthcare provision; the dependence on the Internet of Things (IoT) and cloud-based critical services have all made Cyber Security a critical issue for institutions and governments globally (Maras, 2015; Botta et al., 2016). This has given rise to a growing demand for Cyber Security professionals equipped with the needed skills and knowledge to be able to deal with the rising challenges within cyberspace. This module is in direct response as it is purposefully designed to meet that demand. This module will explore the security implications posed by interconnection and dependence on online services, and how security can be designed and embedded in systems for optimal protection.

Learning Outcomes

I will learn:

  • professionals roles, skillset, ethical responsibilities of Cyber Security professionals
  • Explore the relevant computing architectures, practices, applications, and standards.
  • Develop an understanding of the core concepts in Cyber Security such as cryptography, penetration testing and ethical hacking.
  • Engage with programming concepts necessary for facilitating security-based decisions.
  • Develop the ability to implement a secure information system using the principles and concepts of object-oriented approaches to solve a given problem, such as information system design and web development.
